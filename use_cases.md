# Casos de Uso: Ontología DigiOnto

Para guiar el desarrollo de esta ontología y justificar la integración de nuestras fuentes de datos, hemos definido los siguientes casos de uso:

## 1. Asistente de construcción de equipos competitivos (Team Builder)
Una aplicación dirigida a jugadores de videojuegos de Digimon que permite consultar qué criatura es óptima para un combate. La ontología debe enlazar las estadísticas base de un Digimon (HP, Ataque, Defensa, Velocidad) con su Tipo (Vacuna, Virus, Datos) y Atributo (Fuego, Agua, etc.) para calcular ventajas tácticas.

## 2. Enciclopedia Semántica de Combate
Un portal web donde los usuarios pueden buscar un Digimon y descubrir no solo su etapa evolutiva (Rookie, Champion, Mega), sino también qué ataques (Moves) puede aprender. La ontología permite relacionar la clase principal de la criatura con las habilidades del dataset extendido, filtrando ataques por su coste de SP o poder de daño.
